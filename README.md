# PHP VOCABTRAINER

## What this is

PHP + MySQL vocabulary trainer. Includes a REST API (`/api/vocab`) and three frontend pages (list, quiz, flashcards) intended to be used as iframes. Works locally with XAMPP or any PHP web server. Setup for api and frontend pages is necessary

See the `README.md` files in [frontend](/frontend/README.md) and [api](/api/README.md)
